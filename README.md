# 🎵 Spotify Clone

A responsive **Spotify Frontend Clone** built using **HTML** and **CSS**. This project recreates the look and feel of Spotify's web player with a modern and clean user interface. It is a frontend-only project created to practice HTML, CSS, layout design, and hover animations.

---

## 🚀 Features

* 🎧 Spotify-inspired modern UI
* 📱 Responsive design
* ✨ Smooth hover effects
* 🎨 Clean and organized layout
* 🖼️ Music cards and sidebar navigation
* 💻 Built using only HTML & CSS

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---



## 🎯 Purpose

This project was created to improve my frontend development skills by practicing:

* HTML5 semantic elements
* CSS Flexbox
* Responsive layouts
* Hover animations
* UI cloning
